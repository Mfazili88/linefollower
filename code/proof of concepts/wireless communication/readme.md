### gebruiksaanwijzing
ik schreef de code in arduino IDE en dan de app (Serial Bluetooth Terminal) op mijn smartphone installeerde. ik verbond de bluethooth module met Arduino volgens de schema en dan vond ik de HC-05 op mijn smartphone door zoeken in Bluetooths. dan ging ik naar de Serial Bluetooth Terminal en koos ik in device HC-05. en nu kan ik berichten schrijven en versturen naar de module. ook in serial monitor kan ik de bericht zien of bericht doorsturen naar de Serial Bluetooth Terminal terug.
